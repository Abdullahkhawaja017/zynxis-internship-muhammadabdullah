# Week 2 - Data Preprocessing & Feature Engineering

## Objective

Prepare the Titanic dataset for machine learning by performing preprocessing and feature engineering.

## Tasks Completed

- Handled missing values
- Encoded categorical variables
- Normalized numerical features using StandardScaler
- Removed outliers using the IQR method
- Created two new features:
  - FamilySize
  - IsAlone
- Saved the cleaned dataset

## Files

- Week2_Preprocessing.ipynb
- titanic.csv
- cleaned_titanic.csv
- preprocessing_visualizations/

## Libraries Used

- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Outcome

The dataset has been successfully cleaned and prepared for machine learning. The engineered features and preprocessing steps improve the dataset's quality for future classification and regression models.
